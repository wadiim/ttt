## Building

``` shell
cmake . -Bbuild
cmake --build build
```

## Key bindings

| Keystroke | Effect |
| ------------- | -------- |
| <kbd>Q</kbd> | Quit |
| <kbd>←</kbd> / <kbd>H</kbd> | Move left |
| <kbd>↓</kbd> / <kbd>J</kbd> | Move down |
| <kbd>↑</kbd> / <kbd>K</kbd> | Move up |
| <kbd>→</kbd> / <kbd>L</kbd> | Move right |
| <kbd>Enter</kbd> / <kbd>Spacebar</kbd> | Insert `X` |
